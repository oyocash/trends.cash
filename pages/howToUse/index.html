How to use

Visualize single phrase

Just write the phrase you search for. For example bitcoin sv [see on Trends]

Visualize multiple phrases

To search for multiple multiple phrases, separate them with ','. For example, bsv, abc [see on Trends]

Visualize protocols

Put prefix: in front of a protocol code. For example, to search for Memo posts, use prefix:6d02 [see on Trends]

Visualize apps

Put app: in front of a the app identifier. For example, to search for Twetch actions, use app:twetch [see on Trends]

Visualize transactions from (sender)

Put from: in front of the sender's address. For example from:17yvo2MY6DSX4JFJNWzEdav95EDE9PcWvx [see on Trends]

Visualize transactions excluding sender

Put notfrom: in front of the sender's address. For example notfrom:17yvo2MY6DSX4JFJNWzEdav95EDE9PcWvx [see on Trends]

Visualize transactions to (receiver)

Put to: in front of the receiver's address. For example to:17yvo2MY6DSX4JFJNWzEdav95EDE9PcWvx [see on Trends]

Visualize transactions excluding sender

Put notto: in front of the receiver's address. For example notto:17yvo2MY6DSX4JFJNWzEdav95EDE9PcWvx [see on Trends]

Combining searches

You can combine the methods above as you like. Examples:
prefix:6d02 satoshi (searches for 'satoshi' in memo posts) [see on Trends]
prefix:6d03, prefix:6d04 (compared memo replies and memo posts) [see on Trends]
prefix:6d04 from: 17yvo2MY6DSX4JFJNWzEdav95EDE9PcWvx (visualizes likes on memo by user address) [see on Trends]
